# CutVerse

CutVerse is building a new generation of multimodal creative agent benchmarks and evaluation infrastructure.

## What Is CutVerse

CutVerse focuses on one core mission:
- define realistic creative software tasks,
- measure agent capability reliably,
- and accelerate progress from demo-level interaction to production-grade automation.

## Why It Matters

Multimodal agents still struggle in long-horizon GUI workflows. CutVerse is designed to close that gap by emphasizing:
- cross-modal understanding,
- robust trajectory-based evaluation,
- and reproducible, system-level benchmarking.

## Current Stage

This repository is intentionally a **public-facing intro repo**.
It currently contains no source code release.
